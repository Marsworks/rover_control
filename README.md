# teleop
Marsworks 2020 teleop ROS package

## Requirements
### System Requirements
* Ubuntu 16.04
* ROS Kinect

### ROS Packages
* joy
```shell
sudo apt-get install ros-$ROS_DISTRO-joystick_drivers
sudo apt-get install ros-$ROS_DISTRO-joy
```
* rosserial_python
```shell
sudo apt-get install ros-$ROS_DISTRO-rosserial-python
```
* rospy
